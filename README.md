# ChillScape Frontend

Frontend part of the ChillScape project built with Next.js.

## Tech Stack

- Next.js
- React
- TypeScript
- CSS Modules

## Main Routes

- `/` — Home page
- `/login` — Login page
- `/signup` — Sign up page
- `/locations` — Locations page
- `/locations/add` — Add location page
- `/profile` — Profile page

## Getting Started

Install dependencies:

```bash
npm install
```
